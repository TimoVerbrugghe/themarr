Environment Variables

Variable Default Description
Plex
PLEX_URLrequiredBase URL of your Plex server (e.g. http://192.168.1.100:32400)
PLEX_TOKENrequiredPlex authentication token. Find it via your Plex account settings.
Jellyfin
JELLYFIN_URL—Base URL of your Jellyfin server (e.g. http://192.168.1.50:8096).
JELLYFIN_API_KEY—Jellyfin API key (Dashboard → API Keys).
JELLYFIN_USER_IDfirst userOptional Jellyfin user ID used for library/item access. If omitted, Themarr uses the first user returned by Jellyfin.
Media paths
TV_SHOWS_HOST_PATH—Host filesystem path for TV shows, mounted into the container at the same path as your Plex container.
MOVIES_HOST_PATH—Host filesystem path for movies, mounted into the container at the same path as your Plex container.
Web UI
DEFAULT_THEMEdarkInitial colour theme (dark or light). Users can override this in-browser.
DEFAULT_VIEWlistInitial library view (list or grid). Users can override this in-browser via the view-toggle buttons.
FLASK_DEBUGfalseEnable Flask debug mode. Do not use in production.
API_AUTH_TOKENauto-generatedAPI token used for all mutating API routes. If unset, Themarr generates a startup token and shows it above.
Notifications (optional)
PUSHOVER_APP_TOKEN—Pushover application token. Leave blank to disable push notifications.
PUSHOVER_USER_KEY—Pushover user/group key. Leave blank to disable push notifications.
Webhooks (optional)
WEBHOOK_USERNAME—Basic-auth username for the Plex webhook endpoint.
WEBHOOK_PASSWORD—Basic-auth password for the Plex webhook endpoint.
PLEX_RETRY_ATTEMPTS10Max number of Plex lookup attempts after a webhook event.
PLEX_RETRY_DELAY30Seconds between each Plex lookup attempt (linearly increasing).
